# SevnJS : 7 function library for frontend  
### Formely known as GeneratorJS

This is the main project repo for SevnJS/cdn: A 7 function library for modular frontend design. 

[Website](https://sevnjs.github.io)  
[Github](https://github.com/sevnjs/cdn.git)

#### Main functions
1. [gen()](https://sevnjs.github.io/docs/gen)
2. [append()](https://sevnjs.github.io/docs/append)
3. [load()](https://sevnjs.github.io/docs/load)
4. [log()](https://sevnjs.github.io/docs/log)
5. [loadscss()](https://sevnjs.github.io/docs/loadscss)
6. [getfile()](https://sevnjs.github.io/docs/getfile)
7. [parsemd()](https://sevnjs.github.io/docs/parsemd)

#### Optional functions

- [grab()](https://sevnjs.github.io/docs/grab)
- [getfile()](https://sevnjs.github.io/docs/getfile)
